# Stock Trading with Machine Learning

## Overview

A stock trading bot that uses machine learning to make price predictions. Forked from yacoubb.

## Requirements

-   Python 3.5+
-   alpha_vantage
-   pandas
-   numpy
-   sklearn
-   keras
-   tensorflow
-   matplotlib
